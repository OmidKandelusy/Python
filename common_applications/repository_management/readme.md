NOTE:
This script can be used directly from the terminal using the bundle bash script or
from the IDE like vs-code. However, note that if you use vs-code, since the vscode
caches the virtual environment for the next interpretation of python, if you run this
script from the vscode, you will most likely get error because the cached venv will no
longer exsit at the end of the script cleaning phas. To resolve that issue, you would need
to re-select the python interpreter.
